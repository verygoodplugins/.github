# Very Good Plugins

**Handcrafted WordPress tools and AI-powered developer utilities.**

We build [WP Fusion](https://wpfusion.com/?utm_source=github&utm_medium=profile) and [Fatal Error Notify](https://fatalerrornotify.com/?utm_source=github&utm_medium=profile) for WordPress, plus a growing collection of open-source [MCP servers](https://modelcontextprotocol.io) for AI-assisted workflows.
